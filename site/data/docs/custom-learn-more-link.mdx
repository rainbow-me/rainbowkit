# Custom "Learn more" link

## Customizing the "Learn more" link

By default, the introductory "Learn more" link within the "What is a wallet?" section points to [this article](https://learn.rainbow.me/what-is-a-cryptoweb3-wallet-actually), but you can override this via the `learnMoreUrl` prop on `RainbowKitProvider`.

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => {
  return (
    <RainbowKitProvider learnMoreUrl="https://learn.rainbow.me/what-is-a-cryptoweb3-wallet-actually">
      {/* Your App */}
    </RainbowKitProvider>
  );
};
```
