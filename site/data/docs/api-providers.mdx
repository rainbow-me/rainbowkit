# API Providers

## Customizing API providers

RainbowKit has built-in API provider support so you don't have to worry about defining RPC URLs & a provider instance.

RainbowKit exports `apiProvider` that you can use to instantiate your API provider.

#### Alchemy

To use the built-in Alchemy API provider, you will need to pass through your **chains** & **Alchemy ID**.

```tsx line=1,4-7,11,17
import { apiProvider, getDefaultWallets } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { chains, provider } = apiProvider.alchemy(
  [chain.mainnet, chain.polygon], 
  process.env.ALCHEMY_ID
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
});

...
```

If you wish to use a WebSocket provider, the API Provider also returns a `webSocketProvider`.

```tsx line=4,18
import { apiProvider, getDefaultWallets } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { chains, provider, webSocketProvider } = apiProvider.alchemy(
  [chain.mainnet, chain.polygon], 
  process.env.ALCHEMY_ID
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
  webSocketProvider,
});

...
```

#### Custom

To use a custom API provider, you will need to pass through your **chains** and a map of **RPC URLs** (`rpcUrls`) of the chains you want to support.

```tsx line=1,4-12,16,22
import { apiProvider, getDefaultWallets } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { chains, provider } = apiProvider.custom(
  [chain.mainnet, chain.polygon],
  {
    rpcUrls: {
      [chain.mainnet.id]: 'https://mainnet.example.com',
      [chain.polygon.id]: 'https://polygon.example.com',
    }
  }
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains,
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
});

...
```

If you want to use a WebSocket provider, you can pass through your WebSocket RPC URLs using the `webSocketRpcUrls` option.

```tsx line=4,11-14,27
import { apiProvider, getDefaultWallets } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { chains, provider, webSocketProvider } = apiProvider.custom(
  [chain.mainnet, chain.polygon],
  {
    rpcUrls: {
      [chain.mainnet.id]: 'https://mainnet.example.com',
      [chain.polygon.id]: 'https://polygon.example.com',
    },
    webSocketRpcUrls: {
      [chain.mainnet.id]: 'wss://mainnet.example.com',
      [chain.polygon.id]: 'wss://polygon.example.com',
    }
  }
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains,
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
  webSocketProvider
});

...
```
