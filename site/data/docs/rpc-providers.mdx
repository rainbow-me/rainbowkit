# RPC Providers

## Customizing RPC providers

RainbowKit has built-in RPC provider support so you don't have to worry about defining RPC URLs & a provider instance.

RainbowKit exports `rpcProvider` that you can use to instantiate your RPC provider.

#### Alchemy

To use the built-in Alchemy RPC provider, you will need to pass through your **chains** & **Alchemy ID**.

```tsx line=1,4-7,11,17
import { rpcProvider } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { provider, chains } = rpcProvider.alchemy(
  [chain.mainnet, chain.polygon], 
  process.env.ALCHEMY_ID
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
});

...
```

#### Custom

To use a custom RPC provider, you will need to pass through your **chains** and a map of **RPC URLs** (`rpcUrls`) of the chains you want to support.

```tsx line=1,4-12,16,22
import { rpcProvider } from '@rainbow-me/rainbowkit';
import { chain } from 'wagmi';

const { provider, chains } = rpcProvider.custom(
  [chain.mainnet, chain.polygon],
  {
    rpcUrls: {
      [chain.mainnet.id]: 'https://mainnet.example.com',
      [chain.polygon.id]: 'https://polygon.example.com',
    }
  }
);

const { connectors } = getDefaultWallets({
  appName: 'RainbowKit demo',
  chains,
});

const wagmiClient = createClient({
  autoConnect: true,
  connectors,
  provider,
});

...
```
