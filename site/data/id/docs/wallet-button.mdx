# TombolDompet

> Catatan: `WalletButton` saat ini mengandalkan standar dompet EIP-1193, namun akan mendukung EIP-6963 dalam waktu dekat.

Komponen `TombolDompet` baru membantu pengembang aplikasi dengan implementasi daftar dompet khusus dapat mengadopsi RainbowKit dengan semua keuntungan tanpa perlu pemeliharaan.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Seperti `TombolSambung`, komponen `TombolDompet.Individu` juga tersedia untuk implementasi dan penyesuaian khusus.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Kebanyakan aplikasi terbaik dilayani oleh [ConnectButton](https://www.rainbowkit.com/docs/connect-button). Merujuk ke dokumen [here](https://www.rainbowkit.com/docs/wallet-button) untuk informasi lebih lanjut tentang pengadopsian dan penggunaan `TombolDompet`.
