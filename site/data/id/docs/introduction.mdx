# RainbowKit

<Video src="https://rainbowme-res.cloudinary.com/video/upload/v1657717102/rainbowkit/site-assets/rainbowkit-video-final-3_dehldu.mp4" />

RainbowKit adalah pustaka [React](https://reactjs.org/) yang memudahkan penambahan koneksi dompet ke dapp Anda. Ini intuitif, responsif dan dapat disesuaikan.

### Fitur

#### Manajemen dompet

Manajemen dompet siap pakai untuk dapp Anda. Selain menangani koneksi dan diskoneksi dompet, RainbowKit mendukung berbagai dompet, mengganti koneksi rantai, menjelaskan alamat ke ENS, menampilkan saldo dan masih banyak lagi!

#### Dapat disesuaikan

Anda dapat menyesuaikan UI RainbowKit agar sesuai dengan merek Anda. Anda dapat memilih dari beberapa warna aksen dan konfigurasi radius batas yang telah ditentukan sebelumnya. Untuk kasus penggunaan yang lebih lanjut, Anda dapat memberikan tema kustom sepenuhnya, merender tombol Anda sendiri dan menghilangkan fitur tertentu. Mode Gelap disertakan.
