# Lokalisasi

## Menyesuaikan bahasa dApp Anda dengan terjemahan bawaan

Secara default, RainbowKit mendukung lokal `en-US` untuk pengguna berbahasa Inggris.

Jika tersedia, RainbowKit akan mendeteksi [bahasa yang disukai pengguna](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/language) dan memilih terjemahan yang sesuai. Pengembang selalu dapat mengganti bahasa default.

Untuk menentukan bahasa bagi pengguna Anda, cukup tambahkan `locale="zh-CN"` sebagai prop dalam `RainbowKitProvider` Anda

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider locale="zh-CN" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### Menggunakan dengan Next.js

Dukungan lokalisasi RainbowKit bekerja lebih baik dengan [Sub-path Routing](https://nextjs.org/docs/pages/building-your-application/routing/internationalization#sub-path-routing).

Konfigurasikan proyek Next.js Anda seperti contoh di bawah ini untuk menambahkan sub-path `/locale/` opsional ke rute Anda, yang akan membantu mesin pencari dan pengguna lebih baik menemukan dukungan multibahasa Anda.

```json
// next.config.js
{
  i18n: {
    locales: ['default', 'en', 'zh-CN'],
    defaultLocale: 'default',
  },
}
```

Lalu berikan `locale` yang disediakan oleh Pages Router ke `RainbowKitProvider`

```tsx
import { RainbowKitProvider, Locale } from '@rainbow-me/rainbowkit';

export const App = () => {
  const { locale } = useRouter() as { locale: Locale };
  return (
    <RainbowKitProvider locale={locale} {...etc}>
      {/* Your App */}
    </RainbowKitProvider>
  )
};
```

Anda dapat merujuk contoh untuk Pages Router [here](https://github.com/rainbow-me/rainbowkit/tree/main/examples/with-next).

App Router belum mendukung i18n. Rujuk contoh kami yang terpisah [here](https://github.com/rainbow-me/rainbowkit/tree/main/examples/with-next-app-i18n) untuk praktek terbaik implementasi dengan middleware [`next-intl`](https://github.com/amannn/next-intl).

Disarankan agar Anda menggunakan teknik yang sama untuk menerjemahkan konten dApp Anda untuk dukungan lokalisasi penuh. Pustaka lokal seperti [`i18n-js`](https://github.com/fnando/i18n) dan [`next-intl`](https://github.com/amannn/next-intl) serta alat manajemen seperti Crowdin akan menyederhanakan proses ini.
