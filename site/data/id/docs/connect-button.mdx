# ConnectButton

## Menggunakan dan menyesuaikan ConnectButton

Ini adalah komponen utama. Ini bertanggung jawab untuk merender tombol connect/disconnect, serta antarmuka pindah rantai.

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> Catatan: Pastikan aplikasi Anda dibungkus dalam penyedia yang diperlukan. [Baca lebih lanjut](/docs/installation#wrap-providers).

### Props

Komponen `ConnectButton` mengungkapkan beberapa props untuk menyesuaikan tampilannya, termasuk mengonversi visibilitas elemen yang berbeda.

    },
  ]}
/>

> Catatan: titik patah bawaan `largeScreen` adalah `768px`.

### Contoh

#### Label

Gunakan `label` prop untuk mengatur teks `ConnectButton` yang kustom.

```tsx
<ConnectButton label="__Sign in__" />
```

#### Status akun

Berikut adalah beberapa cara yang bisa Anda gunakan untuk prop `accountStatus`.

Hanya tampilkan avatar akun.

```tsx
<ConnectButton accountStatus="__avatar__" />
```

Hanya tampilkan alamat akun.

```tsx
<ConnectButton accountStatus="__address__" />
```

#### Status rantai

Berikut adalah beberapa cara yang bisa Anda gunakan untuk prop `chainStatus`.

Hanya tampilkan ikon rantai saat ini.

```tsx
<ConnectButton chainStatus="__icon__" />
```

Hanya tampilkan nama rantai saat ini.

```tsx
<ConnectButton chainStatus="__name__" />
```

Sembunyikan seluruh UI rantai.

```tsx
<ConnectButton chainStatus="__none__" />
```

#### Tampilkan saldo

Gunakan prop `showBalance` untuk sembunyikan/menampilkan saldo.

Sembunyikan saldo.

```tsx
<ConnectButton showBalance={__false__} />
```

#### Responsif

Prop ini juga bisa ditentukan dalam format responsif.

Pada layar kecil, hanya tampilkan ikon akun. Tetapi pada layar besar, tampilkan ikon dan alamat.

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

Pada layar kecil, sembunyikan saldo. Tapi di layar besar, tampilkan itu.

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
