# ButangDompet

> Nota: `ButangDompet` pada masa ini bergantung kepada standard dompet EIP-1193, tetapi akan menyokong EIP-6963 pada masa hadapan.

Komponen `ButangDompet` baharu membantu dApps dengan pelaksanaan senarai dompet khusus mengadopsi RainbowKit dan semua faedahnya yang bebas penyelenggaraan.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

Seperti `ButangSambung`, komponen `ButangDompet.Khusus` tersedia untuk pelaksanaan dan penataan khusus.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Kebanyakan dApps lebih baik menggunakan [ButangSambung](https://www.rainbowkit.com/docs/connect-button). Rujuk dokumentasi [di sini](https://www.rainbowkit.com/docs/wallet-button) untuk maklumat lanjut tentang pengadopsian dan kes penggunaan `ButangDompet`.
