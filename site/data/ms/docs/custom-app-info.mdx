# Maklumat Aplikasi Tersuai

## Menyesuaikan maklumat aplikasi anda

Anda boleh memasukkan maklumat aplikasi anda dalam prop `appInfo` untuk `RainbowKitProvider`. Ciri-ciri yang boleh anda ubah ialah nama aplikasi anda (`appName`) dan pautan di mana butang "Pelajari Lebih Lanjut" dalam mod sambungan mengarah (`learnMoreUrl`):

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'Rainbowkit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```

### Tambah Penafian

Anda boleh menyediakan penafian anda sendiri untuk dipaparkan di bahagian bawah mod sambungan. Ciri `disclaimer` dalam `appInfo` mengambil `DisclaimerComponent` dengan argumen `Text` dan `Link`. Anda boleh menggunakannya untuk membina penafian anda sendiri yang mewarisi sistem tema. Kami mengekspor jenis `DisclaimerComponent` untuk pengguna TypeScript:

```tsx
import {
  RainbowKitProvider,
  DisclaimerComponent,
} from '@rainbow-me/rainbowkit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'RainbowKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```
