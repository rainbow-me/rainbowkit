# Senarai Dompet Tersuai

## Menyesuaikan senarai dompet

Anda boleh mengimport dompet individu dari `'@rainbow-me/rainbowkit/wallets'` bersama dengan fungsi `connectorsForWallets` untuk membina senarai dompet anda sendiri dengan penyambung yang diperlukan. Dengan ini, anda mempunyai kawalan penuh ke atas dompet mana yang ingin dipaparkan, dan dalam susunan mana.

Contohnya, anda boleh memilih untuk hanya memaparkan Rainbow bersama dengan dompet jatuh balik generik.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My RainbowKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

Anda kemudian boleh menyampaikan penyambung anda kepada `createConfig` Wagmi.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <RainbowKitProvider {...etc}>
        {/* Your App */}
      </RainbowKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### Dompet Terbina Dalam

Dompet berikut disediakan melalui objek `wallet`.

#### Dompet Generik

Dompet berikut adalah menggunakan kaedah sambungan generik dan bukannya aplikasi khusus. Oleh itu, disarankan agar dompet ini sentiasa disertakan.

##### WalletConnect

Ini adalah pilihan dompet jatuh balik yang direka untuk dompet berasaskan WalletConnect yang tidak disediakan oleh dompet lain dalam senarai.

**Adalah disyorkan agar anda sentiasa menyertakan dompet ini dalam senarai untuk memastikan semua dompet berasaskan WalletConnect disokong.**

```tsx
import { walletConnectWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### Dompet Suntikan

Ini adalah pilihan dompet jatuh balik yang direka untuk senario di mana `window.ethereum` wujud tetapi tidak disediakan oleh dompet lain dalam senarai. Dompet ini akan secara automatik menyembunyikan dirinya daripada senarai apabila jatuh balik tidak diperlukan atau jika tiada dompet suntikan tersedia.

**Adalah disyorkan agar anda sentiasa menyertakan dompet ini dalam senarai untuk memastikan semua dompet suntikan disokong.**

```tsx
import { injectedWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### Safe Wallet

Untuk dApps yang menyokong Safe Apps bagi membenarkan pengguna menghubungkan dengan peti simpanan selamat atau multisig mereka, sertakan pilihan `safeWallet` untuk sambungan automatik. Pilihan Safe akan hanya muncul dalam persekitaran penyemak imbas Safe Wallet.

```tsx
import { safeWallet } from '@rainbow-me/rainbowkit/wallets';
```

#### Coinbase

Penyambung dompet ini menyokong kedua-dua aplikasi dan sambungan Coinbase Wallet, serta Coinbase Smart Wallet dalam Web.

Argumen `preference` tersedia untuk mengawal sama ada Smart Wallet diaktifkan dan tersedia untuk pengguna. Perilaku berasaskan pilihan didokumentasikan [di sini](https://www.smartwallet.dev/sdk/makeWeb3Provider#parameters).

Smart Wallet akan diaktifkan secara lalai dengan `all` pada awal bulan Jun, tanpa peningkatan lanjut.

Pembangun boleh menguji Smart Wallet dengan rantai `sepolia` dan `baseSepolia` hari ini dengan menetapkan `smartWalletOnly` dan memasukkan `coinbaseWallet` dalam senarai dompet mereka seperti berikut:

```tsx
import { coinbaseWallet } from '@rainbow-me/rainbowkit/wallets';

// Enable Coinbase Smart Wallet for testing
coinbaseWallet.preference = 'smartWalletOnly';
```

### Contoh-contoh

#### Aturan

Berikut adalah beberapa contoh memaparkan dompet yang berbeza, dalam aturan yang berbeza.

Paparkan MetaMask bersama dengan dompet jatuh balik generik.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Paparkan Rainbow, MetaMask dan Coinbase bersama dengan dompet jatuh balik generik.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        coinbaseWallet,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> Peringatan: Urutan tatasusunan `wallets` menentukan urutan dompet akan dipaparkan dalam antara muka.

#### Kumpulan Berbilang

Anda boleh menggunakan kunci `groupName` untuk menamakan kumpulan dompet yang berbeza. Ini berguna jika anda ingin memberi pengguna anda maklumat tentang dompet mana yang anda syorkan, serta dompet lain yang mungkin.

Syorkan Rainbow dan MetaMask, tetapi juga tawarkan Coinbase bersama dengan dompet jatuh balik generik.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [coinbaseWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
