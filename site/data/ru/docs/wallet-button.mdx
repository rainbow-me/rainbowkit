# КнопкаПортмонетка

> Примечание: В настоящее время `WalletButton` использует стандарт кошелька EIP-1193, но в ближайшем будущем будет поддерживать EIP-6963.

Новый компонент `КнопкаПортмонетка` помогает децентрализованным приложениям (dApps) с реализацией пользовательского списка кошельков принять RainbowKit и воспользоваться всеми его преимуществами без необходимости поддержки.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Подобно компоненту `КнопкаПодключить`, компонент `КнопкаПортмонетка.Настраиваемая` предназначен для пользовательских реализаций и стилизации.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Большинство dApps лучше всего использовать [ConnectButton](https://www.rainbowkit.com/docs/connect-button). Справка по `КнопкаПортмонетка` и ее применению: [here](https://www.rainbowkit.com/docs/wallet-button).
