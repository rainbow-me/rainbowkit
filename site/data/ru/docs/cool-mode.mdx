# Крутой режим

## Включение крутого режима 🌈🌈🌈

Вы можете включить крутой режим через проп `coolMode` на `RainbowKitProvider`.

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### Но что это?

Эмодзи-взрывы, когда кошелек выбран из списка кошельков 🌈🌈🌈. Он использует иконку выбранного вами кошелька.
