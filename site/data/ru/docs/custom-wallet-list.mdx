# Пользовательский список кошельков

## Настройка списка кошельков

Вы можете импортировать индивидуальные кошельки из `'@rainbow-me/rainbowkit/wallets'` вместе с функцией `connectorsForWallets`, чтобы создать свой собственный список кошельков с их необходимыми соединителями. Таким образом, у вас будет полный контроль над тем, какие кошельки отображать и в каком порядке.

Например, вы можете выбрать отображение только Rainbow вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My RainbowKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

Вы можете затем передать свои соединители в Wagmi `createConfig`.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <RainbowKitProvider {...etc}>
        {/* Your App */}
      </RainbowKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### Встроенные кошельки

Следующие кошельки предоставляются через объект `wallet`.

#### Универсальные кошельки

Следующие кошельки ориентированы на универсальные методы подключения, а не на конкретные приложения. В результате рекомендуется всегда включать эти кошельки.

##### WalletConnect

Это резервный вариант кошелька, предназначенный для кошельков на базе WalletConnect, которые не были предоставлены другим кошельком в списке.

**Рекомендуется всегда включать этот кошелек в список, чтобы обеспечить поддержку всех кошельков на базе WalletConnect.**

```tsx
import { walletConnectWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### Встроенный кошелек

Это резервный вариант кошелька, предназначенный для ситуаций, когда `window.ethereum` существует, но не был предоставлен другим кошельком в списке. Этот кошелек автоматически скроет себя из списка, когда резерв не требуется или если доступного встроенного кошелька нет.

**Рекомендуется всегда включать этот кошелек в список, чтобы обеспечить поддержку всех встроенных кошельков.**

```tsx
import { injectedWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### Безопасный кошелек

Для dApps, которые поддерживают Safe Apps, предоставляя пользователям возможность легко подключаться к их Safe vault или мульти-сиг, включите опцию `safeWallet` для автоматического подключения. Опция Safe будет отображаться только в браузерной среде Safe Wallet.

```tsx
import { safeWallet } from '@rainbow-me/rainbowkit/wallets';
```

#### Coinbase

Этот коннектор поддерживает как приложение и расширение Coinbase Wallet, так и Coinbase Smart Wallet на Web.

Доступен аргумент `preference`, который управляет предпочтением использования Smart Wallet или его отключением для ваших пользователей. Поведение на основе предпочтений задокументировано [здесь](https://docs.base.org/identity/smart-wallet/technical-reference/sdk/#options-optional).

Дополнительные параметры конфигурации для управления [Paymasters](https://docs.base.org/identity/smart-wallet/guides/paymasters) и [Под-счетами](https://docs.base.org/identity/smart-wallet/guides/sub-accounts) можно найти в документации Coinbase Wallet SDK [здесь](https://docs.base.org/identity/smart-wallet/technical-reference/sdk#parameters).

```tsx
import { coinbaseWallet } from '@rainbow-me/rainbowkit/wallets';

// Limit your dApp to only Smart Wallet
coinbaseWallet.preference = 'smartWalletOnly';

// Limit your dApp to the Coinbase Wallet app
coinbaseWallet.preference = 'eoaOnly';

// Configure the Paymaster for gas sponsorship
coinbaseWallet.paymasterUrls: {
  [base.id]: '...'
}

// Enable Sub Accounts
coinbaseWallet.subAccounts: {
  enableAutoSubAccounts: true,
  defaultSpendLimits: {
    ...
  }
}
```

#### Кошелек Gemini

Этот умный коннектор кошелька позволяет безупречно аутентифицироваться через ключи на keys.gemini.com — без загрузок, расширений или API ключей. Просто работает из коробки.

Параметр `appName` необходим для идентификации вашего dApp, а `appIcon` является необязательным для брендинга опыта подключения.

```tsx
import { geminiWallet } from '@rainbow-me/rainbowkit/wallets';

const gemini = geminiWallet({
  appName: 'My App',
  appIcon: 'https://my-app.com/logo.png', // Optional
});
```

### Примеры

#### Заказ

Вот несколько примеров отображения различных кошельков в разном порядке.

Покажите MetaMask вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Покажите Rainbow, MetaMask и Coinbase вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        coinbaseWallet,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> Напоминание: порядок массива `кошельков` определяет порядок, в котором кошельки будут отображаться в интерфейсе.

#### Несколько групп

Вы можете использовать ключ `groupName`, чтобы назвать различные группы кошельков. Это удобно, если вы хотите сообщить пользователям, какие кошельки вы рекомендуете, а также другие возможные кошельки.

Рекомендуйте Rainbow и MetaMask, но также предложите Coinbase вместе с универсальными резервными кошельками.

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [coinbaseWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
