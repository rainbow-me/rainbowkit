# 酷炫模式

## 打开酷炫模式 🌈🌈🌈

您可以通过在`RainbowKitProvider`上的`coolMode`属性来启用酷炫模式。

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 但它是什么呢？

当从钱包列表中选择一个钱包时，会出现emoji爆炸效果 🌈🌈🌈。 它使用你选择的钱包的图标。
