# 自定义应用信息

## 定制您的应用信息

您可以在 `RainbowKitProvider` 的 `appInfo` prop中传递您的应用信息。 您可以在 `RainbowKitProvider` 的 `appInfo` prop中传递您的应用信息。 您可以修改的属性有您的应用名称 (`appName`) 和“了解更多”按钮在连接模态框中的重定向链接 (`learnMoreUrl`)：

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'Rainbowkit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```

### 添加免责声明

您可以提供自己的免责声明，它将显示在连接模态框的底部。 `appInfo` 中的 `disclaimer` 属性接受一个带有 `Text` 和 `Link` 参数的 `DisclaimerComponent`。 您可以使用它来构建自己的免责声明，该声明继承了主题系统。 我们为 TypeScript 用户导出了 `DisclaimerComponent` 类型：

```tsx
import {
  RainbowKitProvider,
  DisclaimerComponent,
} from '@rainbow-me/rainbowkit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'RainbowKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```
