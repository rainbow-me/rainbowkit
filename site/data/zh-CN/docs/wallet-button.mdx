# 钱包按钮

新的 `WalletButton` 组件帮助 dApps 实现自定义钱包列表并采用 RainbowKit 及其无需维护的优势。

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

与 `ConnectButton` 类似，`WalletButton.Custom` 组件可用于自定义实现和样式化。

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```
