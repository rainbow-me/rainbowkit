# WalletButton

> 注意：`WalletButton` 当前依赖 EIP-1193 钱包标准，但将在不久的将来支持 EIP-6963。

新的 `WalletButton` 组件帮助 dApps 采用 RainbowKit 及其所有免维护的好处，包括自定义钱包列表实现。

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

与 `ConnectButton` 类似， `WalletButton.Custom` 组件可用于自定义实现和样式。

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

大多数 dApps 最适合 [ConnectButton](https://www.rainbowkit.com/docs/connect-button)。 请参考[此处的文档](https://www.rainbowkit.com/docs/wallet-button)以获取有关 `WalletButton` 采用和用例的更多信息。
