# RainbowKit

## 连接钱包的最佳方式 🌈

<Video src="https://rainbowme-res.cloudinary.com/video/upload/v1657717102/rainbowkit/site-assets/rainbowkit-video-final-3_dehldu.mp4" />

RainbowKit 是一个 [React]（https://reactjs.org/）库，可以轻松地将钱包连接添加到您的dapp。 它直观、响应迅速且可自定义。 它直观、响应迅速且可自定义。

### 特性

#### 钱包管理

为您的dapp提供开箱即用的钱包管理。 为您的dapp提供开箱即用的钱包管理。 除了处理钱包的连接和断开外，RainbowKit还支持多种钱包，更换连接链，将地址解析为ENS,显示余额等等!

#### 可定制

你可以调整 RainbowKit UI 以匹配你的品牌风格。 你可以从几种预设的突出颜色和边角半径配置中选择。 对于更高级的使用情况，你可以提供一个完全自定义的主题，渲染你自己的按钮并省略某些功能。 包含暗黑模式。

#### 行业标准

为了与大多数产品更好地互操作，我们依赖于 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/) —— 这是空间中最常用的库。
