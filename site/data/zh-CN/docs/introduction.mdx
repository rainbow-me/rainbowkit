# RainbowKit

## 钱包登录的最佳方式 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一个 [React](https://reactjs.org/) 库，可以轻松地将钱包连接添加到您的 dapp。 它直观、响应迅速且可自定义。

### 特性

#### 钱包管理

为您的 dapp 提供开箱即用的钱包管理。 为您的 dapp 提供开箱即用的钱包管理。除了处理钱包的连接和断开外，RainbowKit还支持多种钱包，更换连接链，将地址解析为ENS，显示余额等等!

#### 可定制

您可以调整 RainbowKit UI 以匹配您的品牌。 您可以从几个预定义的重点色彩和边界半径配置中选择。 对于更高级的用例，您可以提供一个完全自定义的主题，渲染您自己的按钮并省略某些功能。 包括暗模式。

#### 行业标准

为了与大多数产品更好地互操作，我们依赖于 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/) —— 这是空间中最常用的库。

钱包标准如 EIP-1193 和 EIP-6963 均可即插即用，使得浏览器钱包用户能够比以往任何时候都更加无缝地将他们的钱包连接到您的 dApp。
