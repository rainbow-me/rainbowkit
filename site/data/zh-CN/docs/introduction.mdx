# RainbowKit

## 钱包登录的最佳方式 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一个 [React](https://reactjs.org/) 库，可以轻松为您的 dapp 添加钱包连接。它直观、响应迅速且可自定义。

### 功能

#### 钱包管理

为您的 dapp 提供开箱即用的钱包管理。除了处理钱包连接和断开连接，RainbowKit 支持众多钱包、可切换连接链、将地址解析为 ENS、显示余额等多项功能！

#### 可定制

您可以调整 RainbowKit UI 以匹配您的品牌。可以从几个预定义的重点颜色和边框半径配置中选择。对于更高级的用例，您可以提供一个完全自定义的主题，渲染自己的按钮并省略某些功能。包含黑暗模式。

#### 行业标准

为了与大多数产品更好地互操作，我们依赖于 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/) — 在该领域中最常用的库。

像 EIP-1193 和 EIP-6963 这样的钱包标准开箱即用，使浏览器钱包用户比以往更容易无缝连接钱包到您的 dApp。
