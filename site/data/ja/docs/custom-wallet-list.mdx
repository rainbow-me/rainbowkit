# カスタムウォレットリスト

## ウォレットリストのカスタマイズ

必要なコネクタを持つウォレットのリストを構築するために、`'@rainbow-me/rainbowkit/wallets'` から個別のウォレットと `connectorsForWallets` 関数をインポートできます。この方法で、どのウォレットをどの順番で表示するかを完全に制御できます。

例えば、Rainbow と一般的なフォールバックウォレットのみを表示することができます。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, walletConnectWallet],
    },
  ],
  {
    appName: 'My RainbowKit App',
    projectId: 'YOUR_PROJECT_ID',
  }
);
```

その後、Wagmi の `createConfig` にコネクタを渡すことができます。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import { createConfig } from 'wagmi';

const connectors = connectorsForWallets(/* ... */);

const config = createConfig({
  connectors,
  {/* Wagmi config */}
});

const queryClient = new QueryClient();

const App = () => (
  <WagmiProvider config={config}>
    <QueryClientProvider client={queryClient}>
      <RainbowKitProvider {...etc}>
        {/* Your App */}
      </RainbowKitProvider>
    </QueryClientProvider>
  </WagmiProvider>
);
```

### 内蔵ウォレット

以下のウォレットは `wallet` オブジェクトを介して提供されます。

#### 一般ウォレット

以下のウォレットは特定のアプリではなく、一般的な接続方法に対応しています。そのため、これらのウォレットは常に含めることをお勧めします。

##### ウォレットコネクト

これは、他のウォレットリストに提供されていないウォレットコネクトベースのウォレットのために設計されたフォールバックウォレットオプションです。

**ウォレットコネクトベースのすべてのウォレットがサポートされるように、常にこのウォレットをリストに含めることをお勧めします。**

```tsx
import { walletConnectWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### インジェクテッドウォレット

これは、`window.ethereum` が存在するが他のウォレットリストに提供されていないシナリオのために設計されたフォールバックウォレットオプションです。このウォレットは、フォールバックが必要でないか、インジェクテッドウォレットが利用できない場合にはリストから自動的に非表示になります。

**すべてのインジェクテッドウォレットがサポートされるように、常にこのウォレットをリストに含めることをお勧めします。**

```tsx
import { injectedWallet } from '@rainbow-me/rainbowkit/wallets';
```

##### セーフウォレット

SafeアプリをサポートするdAppsには、ユーザーがSafeボールトやマルチシグに簡単に接続できるように`safeWallet`オプションを含めてください。SafeオプションはSafeウォレットブラウザ環境にのみ表示されます。

```tsx
import { safeWallet } from '@rainbow-me/rainbowkit/wallets';
```

#### Coinbase

このウォレットコネクタは、Coinbaseウォレットアプリと拡張機能、およびWeb上でのCoinbaseスマートウォレットをサポートします。

`preference` 引数を使用して、ユーザーに対してスマートウォレットを推奨するか、無効にするかを制御できます。優先度に基づく動作は[こちら](https://docs.base.org/identity/smart-wallet/technical-reference/sdk/#options-optional)に文書化されています。

[Paymasters](https://docs.base.org/identity/smart-wallet/guides/paymasters)および[サブアカウント](https://docs.base.org/identity/smart-wallet/guides/sub-accounts)の制御に関する追加の構成オプションは、CoinbaseウォレットSDKドキュメントの[こちら](https://docs.base.org/identity/smart-wallet/technical-reference/sdk#parameters)にあります。

```tsx
import { coinbaseWallet } from '@rainbow-me/rainbowkit/wallets';

// Limit your dApp to only Smart Wallet
coinbaseWallet.preference = 'smartWalletOnly';

// Limit your dApp to the Coinbase Wallet app
coinbaseWallet.preference = 'eoaOnly';

// Configure the Paymaster for gas sponsorship
coinbaseWallet.paymasterUrls: {
  [base.id]: '...'
}

// Enable Sub Accounts
coinbaseWallet.subAccounts: {
  enableAutoSubAccounts: true,
  defaultSpendLimits: {
    ...
  }
}
```

#### ジェミニウォレット

このスマートウォレットコネクターは、keys.gemini.comでパスキーを通じたシームレスな認証を可能にします。ダウンロード、拡張機能、APIキーは不要です。そのまま機能します。

const gemini = geminiWallet({
  appName: 'My App',
  appIcon: 'https://my-app.com/logo.png', // Optional
});

### 例

#### 順序

異なるウォレットを異なる順序で表示する例をいくつか示します。

MetaMask と一般的なフォールバックウォレットを表示します。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  metaMaskWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [metaMaskWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

Rainbow、MetaMask、およびCoinbaseを一般的なフォールバックウォレットとともに表示します。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Suggested',
      wallets: [
        rainbowWallet,
        metaMaskWallet,
        coinbaseWallet,
        walletConnectWallet,
      ],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```

> 注：`wallets` 配列の順序は、UI に表示されるウォレットの順序を決定します。

#### 複数のグループ

`groupName` キーを使用して、異なるウォレットグループに名前を付けることができます。これにより、ユーザーに推奨するウォレットや他の可能なウォレットを伝えることができます。

Rainbow と MetaMask を推奨し、Coinbase も一般的なフォールバックウォレットとともに提供します。

```tsx
import { connectorsForWallets } from '@rainbow-me/rainbowkit';
import {
  rainbowWallet,
  metaMaskWallet,
  coinbaseWallet,
  walletConnectWallet,
} from '@rainbow-me/rainbowkit/wallets';

const connectors = connectorsForWallets(
  [
    {
      groupName: 'Recommended',
      wallets: [rainbowWallet, metaMaskWallet],
    },
    {
      groupName: 'Others',
      wallets: [coinbaseWallet, walletConnectWallet],
    },
  ],
  { appName: 'RainbowKit App', projectId: 'YOUR_PROJECT_ID' },
);
```
