# RainbowKit

## ウォレットを接続する最良の方法 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit は、dApp にウォレット接続を簡単に追加できる [React](https://reactjs.org/) ライブラリです。直感的、応答性が高く、カスタマイズ可能です。

### 特徴

#### ウォレット管理

dApp のためのワンタッチで使えるウォレット管理。ウォレットの接続と切断の処理だけでなく、RainbowKit は、多くのウォレットに対応し、接続チェーンを交換し、アドレスをENSに解決し、残高を表示するなど、さまざまな機能をサポートしています。

#### カスタマイズ可能

RainbowKit のUIをカスタマイズして、ブランドに合ったスタイルに変更できます。いくつかのあらかじめ定義されたアクセントカラーや境界線の半径の設定から選べます。より高度な使用例として、完全にカスタムテーマを提供し、独自のボタンをレンダリングし、特定の機能を省略することができます。ダークモードを含みます。

#### 業界標準

多くの製品との相互運用性を向上させるために、[viem](https://viem.sh) と [wagmi](https://wagmi.sh/) — 最も一般的に使用されているライブラリに依存しています。

EIP-1193とEIP-6963のようなウォレット基準が、すぐに利用できるので、ブラウザウォレットユーザーがあなたのdAppにシームレスにウォレットを接続することがこれまで以上に簡単になりました。
