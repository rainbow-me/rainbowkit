# Modal Boyutları

## Geniş vs. Kompakt boyut

Varsayılan olarak, RainbowKit'in `modalSize` değeri `wide` olarak ayarlanmıştır (aşağıdaki gibi görünür):

<Img src=\"/modal-default.png\" height=\"569\" width=\"1029\" />

Ayrıca, aşağıdakine benzeyen bir `compact` modal boyutu da sağlıyoruz:

<Img src=\"/modal-compact.png\" height=\"569\" width=\"1029\" />

Modunuzun boyutunu `compact` olarak ayarlamak için `RainbowKitProvider`'ınıza `modalSize="compact"` parametresini ekleyin:

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
