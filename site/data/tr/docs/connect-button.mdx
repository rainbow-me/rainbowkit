# ConnectButton

## ConnectButton'ı kullanma ve özelleştirme

Bu ana bileşendir. Bağlan/Bağlantıyı kes butonunu ve zincir değiştirme arayüzünü render etmekten sorumludur.

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> Not: Uygulamanızın gerekli sağlayıcılarla sarılı olduğundan emin olun. [Daha fazla oku](/docs/installation#wrap-providers).

### Props

`ConnectButton` bileşeni, görünümünü özelleştirmek için çeşitli props'ları açığa çıkarır, farklı öğelerin görünürlüğünü değiştirmeyi içeren.

<PropsTable
  data={[
    {
      name: 'label',
      required: false,
      default: 'Cüzdan Bağla',
      type: 'string',
      description: 'Bağla düğmesinde görünen metin etiketi',
    },
    {
      name: 'accountStatus',
      required: false,
      default: 'tam',
      type: '"avatar" | "adres" | "tam" | { küçükEkran: HesapDurumu, büyükEkran?: HesapDurumu }',
      typeSimple: 'enum',

        'Aktif hesabın avatarı ve/veya adresi gösteriliyor mu',
    },
    {
      name: 'chainStatus',
      required: false,
      default: '{ küçükEkran: "ikon", büyükEkran: "tam" }',
      type: '"ikon" | "isim" | "tam" | "hiçbiri" | { küçükEkran: ZincirDurumu, büyükEkran?: ZincirDurumu }',
      typeSimple: 'enum',
      description:
        'Mevcut zincirin ikonu ve/veya adı görüntüleniyor ya da tamamen gizleniyor mu',
    },
    {
      name: 'showBalance',
      required: false,
      default: '{ küçükEkran: false, büyükEkran: true }',

        'Hesap adının yanında bakiye görüntüleniyor mu',
    },
  ]}
/>

> Not: Yerleşik `largeScreen` kırılma noktası `768px`'dir.

### Örnekler

#### Etiket

Özel bir `ConnectButton` metni ayarlamak için `label` props'unu kullanın.

```tsx
<ConnectButton label="__Sign in__" />
```

#### Hesap durumu

`accountStatus` props'unu farklı şekillerde kullanabileceğiniz bazı yollar.

Sadece hesabın avatarını göster.

```tsx
<ConnectButton accountStatus="__avatar__" />
```

Sadece hesabın adresini göster.

```tsx
<ConnectButton accountStatus="__address__" />
```

#### Zincir durumu

`chainStatus` props'unu farklı şekillerde kullanabileceğiniz bazı yollar.

Sadece mevcut zincirin ikonunu göster.

```tsx
<ConnectButton chainStatus="__icon__" />
```

Sadece mevcut zincirin adını göster.

```tsx
<ConnectButton chainStatus="__name__" />
```

Zincir arayüzünü tamamen gizle.

```tsx
<ConnectButton chainStatus="__none__" />
```

#### Bakiyeyi göster

Bakiyeyi gizlemek/göstermek için `showBalance` props'unu kullanın.

Bakiyeyi gizle.

```tsx
<ConnectButton showBalance={__false__} />
```

#### Duyarlı

Bu props'lar aynı zamanda duyarlı bir formatta tanımlanabilir.

Küçük ekranlarda, yalnızca hesap ikonunu göster. Fakat büyük ekranlarda, ikon ve adresi göster.

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

Küçük ekranlarda, bakiyeyi gizle. Fakat büyük ekranlarda, göster.

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
