# Cüzdan Butonu

> Not: `WalletButton` şu anda EIP-1193 cüzdan standardına dayanmaktadır, ancak yakında EIP-6963'ü destekleyecektir.

Yeni `WalletButton` bileşeni, özel cüzdan listesi uygulamalarına sahip dApp'lerin RainbowKit'e geçmesini ve bakım gerektirmeyen tüm avantajlarından yararlanmasını sağlar.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

`ConnectButton` gibi, `WalletButton.Custom` bileşeni, özel uygulamalar ve stil vermek için kullanılabilir.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Çoğu dApp, [ConnectButton](https://www.rainbowkit.com/docs/connect-button) ile en iyi şekilde çalışır. `WalletButton`'ın benimsenmesi ve kullanım durumları hakkında daha fazla bilgi için [buradaki](https://www.rainbowkit.com/docs/wallet-button) belgelere bakın.
