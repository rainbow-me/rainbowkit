# dApp'ınızda Rainbow Butonu'nu benimsemek

`RainbowButton` bileşeni, dApp'lerinize `wagmi` kullanan ve [RainbowKit](https://www.rainbowkit.com/docs/installation) üzerinde daha özel bir bağlayıcı deneyimini tercih eden kullanıcılar için Rainbow Cüzdan desteği eklemenin en basit yoludur.

**1. `@rainbow-me/rainbow-button` ve eş bağımlılıklarını yükleyin**

Paket, Next.js, React ve Vite ile uyumludur. Eş bağımlılık uyarılarına uyduğunuzdan emin olun.

```bash
npm install @rainbow-me/rainbow-button wagmi viem@2.x @tanstack/react-query
```

**2. Wagmi ile yapılandırın ve RainbowButton'u yükleyin**

`rainbowConnector` örneğini, `projectId` ve `appName` ile Wagmi'nin `createConfig`ine aktarın ve uygulamanızı `RainbowButtonProvider` ile sarmalayın. Ardından `RainbowButton` bileşenini yerleştirin.

```tsx
import '@rainbow-me/rainbow-button/styles.css';
import {
  RainbowButtonProvider,
  RainbowButton,
  rainbowConnector,
} from '@rainbow-me/rainbow-button';
import { WagmiProvider, createConfig, http } from 'wagmi';
import { mainnet } from 'viem/chains';
import {
  QueryClientProvider,
  QueryClient,
} from "@tanstack/react-query";

const config = createConfig({
  connectors: [
    rainbowConnector({
      appName: 'RainbowKit demo',
      projectId: 'YOUR_PROJECT_ID',
    }),
  ],
  chains: [mainnet],
  transports: {
    [mainnet.id]: http(),
  },
  ssr: true,
});

const queryClient = new QueryClient();

function MyApp({ Component, pageProps }: AppProps) {
  return (
    <WagmiProvider config={config}>
      <QueryClientProvider client={queryClient}>
        <RainbowButtonProvider>
          <RainbowButton />
        </RainbowButtonProvider>
      </QueryClientProvider>
    </WagmiProvider>
  );
}
```

Özel uygulamalar ve stil için `RainbowButton.Custom` bileşenini de kullanabilirsiniz.

```tsx
<RainbowButton.Custom>
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</RainbowButton.Custom>
```

**3. Ve bu kadar!**

Artık kullanıcılarınız Rainbow için sorunsuz bir bağlantı deneyiminin keyfini çıkarabilir — bakım veya baş ağrısı olmadan.

Ek cüzdanlar için destek benimsemek istiyorsanız, [RainbowKit](https://www.rainbowkit.com/docs/installation) içinde [`WalletButton`](https://www.rainbowkit.com/docs/wallet-button) bileşeni de mevcuttur.
