# NútVí

> Lưu ý: `NútVí` hiện phụ thuộc vào tiêu chuẩn ví EIP-1193, nhưng sẽ hỗ trợ EIP-6963 trong tương lai gần.

Thành phần `NútVí` mới giúp các dApp với triển khai danh sách ví tùy chỉnh dễ dàng áp dụng RainbowKit và tất cả lợi ích không cần bảo trì của nó.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

Giống như `NútKếtNối`, thành phần `NútVí.TùyChỉnh` cũng có sẵn cho các triển khai và tạo kiểu tùy chỉnh.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Hầu hết các dApp thích hợp nhất với [NútKếtNối](https://www.rainbowkit.com/docs/connect-button). Tham khảo tài liệu [ở đây](https://www.rainbowkit.com/docs/wallet-button) để biết thêm thông tin về sự áp dụng và các trường hợp sử dụng của `NútVí`.
