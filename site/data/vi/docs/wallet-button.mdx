# Nút Ví

> Lưu ý: Hiện tại `Nút Ví` dựa trên tiêu chuẩn ví EIP-1193, nhưng sẽ hỗ trợ EIP-6963 trong tương lai gần.

Thành phần mới `Nút Ví` giúp các dApp với các triển khai danh sách ví tùy chỉnh áp dụng RainbowKit và tất cả các lợi ích không phải bảo trì của nó.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Giống như `Nút Kết Nối`, thành phần `Nút Ví.Tùy Chỉnh` có sẵn cho các triển khai và kiểu dáng tùy chỉnh.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Hầu hết các dApp đều phục vụ tốt nhất bằng [Nút Kết Nối](https://www.rainbowkit.com/docs/connect-button). Tham khảo tài liệu [tại đây](https://www.rainbowkit.com/docs/wallet-button) để biết thêm thông tin về việc áp dụng và sử dụng `Nút Ví`.
