# Chế Độ Cool

## Bật chế độ cool 🌈🌈🌈

Bạn có thể bật chế độ cool thông qua thuộc tính `coolMode` trên `RainbowKitProvider`.

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### Nhưng nó là gì?

Hiệu ứng nổ emoji khi một ví được chọn từ danh sách ví 🌈🌈🌈. Nó sử dụng biểu tượng của ví mà bạn chọn.
