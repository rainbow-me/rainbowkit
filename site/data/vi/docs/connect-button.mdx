# Nút Kết Nối

## Sử dụng và tùy chỉnh Nút Kết Nối

Đây là thành phần chính. Nó chịu trách nhiệm vẽ nút kết nối/ngắt kết nối, cũng như giao diện trao đổi chuỗi.

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> Lưu ý: Đảm bảo ứng dụng của bạn được bao trong nhà cung cấp cần thiết. [Đọc thêm](/docs/installation#wrap-providers).

### Thuộc tính

Thành phần `ConnectButton` cung cấp nhiều thuộc tính để tùy chỉnh giao diện của nó, bao gồm bật/tắt hiển thị các phần tử khác nhau.

<PropsTable
  data={[
    {
      name: 'label',
      required: false,
      default: 'Kết nối Ví',
      type: 'string',
      description: 'Nhãn văn bản hiển thị trên nút Kết nối',
    },
    {
      name: 'accountStatus',
      required: false,
      default: 'đầy đủ',
      type: '"avatar" | "address" | "full" | { smallScreen: AccountStatus, largeScreen?: AccountStatus }',
      typeSimple: 'enum',
      description:
        'Liệu hình đại diện và/hoặc địa chỉ của tài khoản đang hoạt động có được hiển thị hay không',
    },
    {
      name: 'chainStatus',
      required: false,
      default: '{ smallScreen: "icon", largeScreen: "full" }',
      type: '"icon" | "name" | "full" | "none" | { smallScreen: ChainStatus, largeScreen?: ChainStatus }',
      typeSimple: 'enum',
      description:
        "Liệu biểu tượng và/hoặc tên của chuỗi hiện tại được hiển thị hay bị ẩn hoàn toàn",
    },
    {
      name: 'showBalance',
      required: false,
      default: '{ smallScreen: false, largeScreen: true }',
      type: 'boolean | { smallScreen: boolean, largeScreen?: boolean }',
      typeSimple: 'enum',
      description:
        'Liệu số dư có được hiển thị bên cạnh tên tài khoản',
    },
  ]}
/>

> Lưu ý: điểm ngắt `largeScreen` tích hợp là `768px`.

### Ví dụ

#### Nhãn

Sử dụng thuộc tính `label` để đặt văn bản tuỳ chỉnh cho `ConnectButton`.

```tsx
<ConnectButton label="__Sign in__" />
```

#### Trạng thái tài khoản

Đây là một vài cách khác nhau bạn có thể sử dụng thuộc tính `accountStatus`.

Chỉ hiển thị ảnh đại diện của tài khoản.

```tsx
<ConnectButton accountStatus="__avatar__" />
```

Chỉ hiển thị địa chỉ của tài khoản.

```tsx
<ConnectButton accountStatus="__address__" />
```

#### Trạng thái chuỗi

Đây là một vài cách khác nhau bạn có thể sử dụng thuộc tính `chainStatus`.

Chỉ hiển thị biểu tượng chuỗi hiện tại.

```tsx
<ConnectButton chainStatus="__icon__" />
```

Chỉ hiển thị tên chuỗi hiện tại.

```tsx
<ConnectButton chainStatus="__name__" />
```

Ẩn hoàn toàn giao diện chuỗi.

```tsx
<ConnectButton chainStatus="__none__" />
```

#### Hiển thị số dư

Sử dụng thuộc tính `showBalance` để ẩn/hiển thị số dư.

Ẩn số dư.

```tsx
<ConnectButton showBalance={__false__} />
```

#### Đáp ứng

Các thuộc tính này cũng có thể được định nghĩa ở định dạng đáp ứng.

Trên màn hình nhỏ, chỉ hiển thị biểu tượng tài khoản. Nhưng trên màn hình lớn, hiển thị biểu tượng và địa chỉ.

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

Trên màn hình nhỏ, ẩn số dư. Nhưng trên màn hình lớn, hiển thị nó.

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
