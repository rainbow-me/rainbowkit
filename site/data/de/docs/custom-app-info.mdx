# Benutzerdefinierte App-Informationen

## Anpassen der Informationen Ihrer App

Sie können die Informationen Ihrer App in der `appInfo` Eigenschaft des `RainbowKitProvider` übergeben. Eigenschaften, die Sie ändern können, sind der Name Ihrer App (`appName`) und der Link, auf den die Schaltfläche „Mehr erfahren“ im Verbindungsdialogfeld weiterleitet (`learnMoreUrl`):

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'Rainbowkit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```

### Einen Haftungsausschluss hinzufügen

Sie können Ihren eigenen Haftungsausschluss bereitstellen, der unten im Verbindungsdialogfeld angezeigt wird. Die `disclaimer` Eigenschaft in `appInfo` nimmt ein `DisclaimerComponent` mit den Argumenten `Text` und `Link` an. Sie können es verwenden, um Ihren eigenen Haftungsausschluss zu erstellen, der das Thematisierungssystem erbt. Wir exportieren den `DisclaimerComponent` Typ für TypeScript-Nutzer:

```tsx
import {
  RainbowKitProvider,
  DisclaimerComponent,
} from '@rainbow-me/rainbowkit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'RainbowKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```
