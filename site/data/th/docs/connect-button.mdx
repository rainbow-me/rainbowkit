# ConnectButton

## การใช้และปรับแต่ง ConnectButton

นี่คือส่วนประกอบหลัก มีหน้าที่ในการแสดงผลปุ่มเชื่อมต่อ/ยกเลิกการเชื่อมต่อ รวมถึง UI การสลับ chain

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> หมายเหตุ: ตรวจสอบให้แน่ใจว่าแอปของคุณถูกห่อด้วยผู้ให้บริการที่จำเป็น [อ่านเพิ่มเติม](/docs/installation#wrap-providers).

### พร็อพ

องค์ประกอบ `ConnectButton` เผยพร็อพหลายตัวให้กำหนดลักษณะของมัน ซึ่งรวมถึงการแสดง/ซ่อนขององค์ประกอบต่างๆ

### ตัวอย่าง

#### ป้ายชื่อ

ใช้พร็อพ `label` เพื่อกำหนดข้อความ ConnectButton เอง

```tsx
<ConnectButton label="__Sign in__" />
```

#### สถานะบัญชี

นี่คือบางวิธีที่คุณสามารถใช้พร็อพ `accountStatus`

แสดงเฉพาะอวตารของบัญชี

```tsx
<ConnectButton accountStatus="__avatar__" />
```

แสดงเฉพาะที่อยู่ของบัญชี

```tsx
<ConnectButton accountStatus="__address__" />
```

#### สถานะ chain

นี่คือบางวิธีที่คุณสามารถใช้พร็อพ `chainStatus`

แสดงเฉพาะไอคอนของ chain ปัจจุบัน

```tsx
<ConnectButton chainStatus="__icon__" />
```

แสดงเฉพาะชื่อของ chain ปัจจุบัน

```tsx
<ConnectButton chainStatus="__name__" />
```

ซ่อน UI ของ chain ทั้งหมด

```tsx
<ConnectButton chainStatus="__none__" />
```

#### แสดงยอดเงิน

ใช้พร็อพ `showBalance` เพื่อซ่อน/แสดงยอดเงิน

ซ่อนยอดเงิน

```tsx
<ConnectButton showBalance={__false__} />
```

#### การตอบสนอง

พร็อพเหล่านี้ยังสามารถกำหนดในรูปแบบการตอบสนองได้

บนหน้าจอขนาดเล็ก แสดงเฉพาะไอคอนบัญชี แต่ในหน้าจอขนาดใหญ่ แสดงไอคอนและที่อยู่

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

บนหน้าจอขนาดเล็ก ซ่อนยอดเงิน แต่บนหน้าจอขนาดใหญ่ แสดงมัน

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
