# ConnectButton

## การใช้และปรับแต่ง ConnectButton

นี่คือหลักสำคัญ. มันมีความรับผิดชอบในการแสดงปุ่มเชื่อมต่อ/หักการเชื่อมต่อ รวมถึง UI สำหรับสลับเชื่อมโยง.

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> หมายเหตุ: ตรวจสอบให้แน่ใจว่าแอพของคุณมีการห่อผู้ให้บริการที่จำเป็น. [อ่านเพิ่มเติม](/docs/installation#wrap-providers).

### Props

ปุ่ม `ConnectButton` จะเปิดให้คุณปรับแต่งลักษณะในหลายๆ รายการ รวมถึงการสลับการมองเห็นขององค์ประกอบต่างๆ.

    },

> หมายเหตุ: จุดช่วง `largeScreen` ในตัวจะเป็น `768px`.

### ตัวอย่าง

#### ป้าย

ใช้ prop `label` เพื่อกำหนดข้อความ `ConnectButton` ที่กำหนดเอง.

```tsx
<ConnectButton label="__Sign in__" />
```

#### สถานะบัญชี

นี่คือวิธีที่แตกต่างกันหลายวิธีที่คุณสามารถใช้ `accountStatus` prop.

แสดงเฉพาะอวาตาร์ของบัญชี.

```tsx
<ConnectButton accountStatus="__avatar__" />
```

แสดงเฉพาะที่อยู่ของบัญชี.

```tsx
<ConnectButton accountStatus="__address__" />
```

#### สถานะของ Chain

นี่คือวิธีที่แตกต่างกันหลายวิธีที่คุณสามารถใช้ `chainStatus` prop.

แสดงเฉพาะไอคอนของ chain ปัจจุบัน.

```tsx
<ConnectButton chainStatus="__icon__" />
```

แสดงเฉพาะชื่อของ chain ปัจจุบัน.

```tsx
<ConnectButton chainStatus="__name__" />
```

ซ่อนหน้าตาของ chain ทั้งหมด.

```tsx
<ConnectButton chainStatus="__none__" />
```

#### แสดงยอดเงิน

ใช้ `showBalance` prop เพื่อซ่อน/แสดงยอดเงิน.

ซ่อนยอดเงิน.

```tsx
<ConnectButton showBalance={__false__} />
```

#### Responsiveness

props นี้สามารถกำหนดได้โดยรูปแบบที่เต็จตอบสนอง.

บนจอที่มีขนาดเล็ก, แสดงเฉพาะไอคอนบัญชี. แต่บนจอที่มีขนาดใหญ่, แสดงไอคอนและที่อยู่.

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

บนจอที่มีขนาดเล็ก, ซ่อนยอดเงิน. แต่บนหน้าจอขนาดใหญ่ แสดงมัน

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
