# ปุ่มกระเป๋าเงิน

> หมายเหตุ: `ปุ่มกระเป๋าเงิน` ปัจจุบันอาศัยมาตรฐานกระเป๋าเงิน EIP-1193 แต่จะรองรับ EIP-6963 ในอนาคตอันใกล้

คอมโพแนนต์ใหม่ `ปุ่มกระเป๋าเงิน` ช่วยให้ dApps ใช้การนำเสนอรายการกระเป๋าเงินแบบกำหนดเองเพื่อยอมรับ RainbowKit และประโยชน์ที่ปราศจากการบำรุงรักษาทั้งหมด

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

เหมือนกับ `ConnectButton`, คอมโพแนนต์ `WalletButton.Custom` พร้อมสำหรับการดำเนินการและการออกแบบแบบกำหนดเอง

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

ส่วนใหญ่ของ dApp จะได้รับประโยชน์มากที่สุดจาก [ConnectButton](https://www.rainbowkit.com/docs/connect-button) อ้างอิงเอกสาร [here](https://www.rainbowkit.com/docs/wallet-button) เพื่อข้อมูลเพิ่มเติมเกี่ยวกับการใช้งานและการนำเอา `WalletButton` มาใช้ในฐานะตัวดำเนินการ อ้างอิงเอกสาร [here](https://www.rainbowkit.com/docs/wallet-button) เพื่อข้อมูลเพิ่มเติมเกี่ยวกับการใช้งานและการนำเอา `WalletButton` มาใช้ในฐานะตัวดำเนินการ
