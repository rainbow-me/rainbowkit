# WalletButton

> หมายเหตุ: ปัจจุบัน `WalletButton` อิงตามมาตรฐานกระเป๋าเงิน EIP-1193 แต่อยู่ในขั้นตอนการสนับสนุน EIP-6963 ในอนาคตอันใกล้

คอมโพเนนต์ `WalletButton` ใหม่ช่วยให้ dApps ที่มีการใช้งานรายการกระเป๋าเงินแบบกำหนดเองสามารถนำ RainbowKit มาใช้และได้รับประโยชน์ที่ไม่ต้องบำรุงรักษา

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

เช่นเดียวกับ `ConnectButton`, คอมโพเนนต์ `WalletButton.Custom` สามารถใช้สำหรับการปรับแต่งและการออกแบบตามเองได้

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

dApps ส่วนใหญ่ให้บริการดีที่สุดด้วย [ConnectButton](https://www.rainbowkit.com/docs/connect-button) ดูเอกสาร [ที่นี่](https://www.rainbowkit.com/docs/wallet-button) สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการนำไปใช้และกรณีการใช้ `WalletButton`
