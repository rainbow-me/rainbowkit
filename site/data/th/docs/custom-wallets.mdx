# กระเป๋าสตางค์ที่กำหนดเอง

## สร้างกระเป๋าสตางค์ที่กำหนดเอง

> หมายเหตุ: API นี้ยังไม่เสถียรและอาจมีการเปลี่ยนแปลงในอนาคตอันใกล้ เราจะเพิ่มกระเป๋าสตางค์ในตัวที่มีอยู่ในเวลาที่กำหนด [แจ้งให้เราทราบ](https://github.com/rainbow-me/rainbowkit/issues) หากคุณสนใจในกระเป๋าสตางค์ใดเป็นพิเศษ 

ฟังก์ชัน `Wallet` ถูกเสนอขึ้นมาเพื่อช่วยให้คุณสามารถกำหนดกระเป๋าสตางค์ที่กำหนดเองของคุณเองได้ สามารถกำหนดคุณสมบัติต่อไปนี้ในผลลัพธ์ของฟังก์ชัน `Wallet`:

### ตัวอย่าง

ตัวอย่างเช่น เพื่อสร้างกระเป๋าสตางค์ที่กำหนดเองโดยใช้ WalletConnect:

```tsx
import { Wallet, getWalletConnectConnector } from '@rainbow-me/rainbowkit';

export interface MyWalletOptions {
  projectId: string;
}

export const rainbow = ({ projectId }: MyWalletOptions): Wallet => ({
  id: 'my-wallet',
  name: 'My Wallet',
  iconUrl: 'https://my-image.xyz',
  iconBackground: '#0c2f78',
  downloadUrls: {
    android: 'https://play.google.com/store/apps/details?id=my.wallet',
    ios: 'https://apps.apple.com/us/app/my-wallet',
    chrome: 'https://chrome.google.com/webstore/detail/my-wallet',
    qrCode: 'https://my-wallet/qr',
  },
  mobile: {
    getUri: (uri: string) => uri,
  },
  qrCode: {
    getUri: (uri: string) => uri,
    instructions: {
      learnMoreUrl: 'https://my-wallet/learn-more',
      steps: [
        {
          description:
            'We recommend putting My Wallet on your home screen for faster access to your wallet.',
          step: 'install',
          title: 'Open the My Wallet app',
        },
        {
          description:
            'After you scan, a connection prompt will appear for you to connect your wallet.',
          step: 'scan',
          title: 'Tap the scan button',
        },
      ],
    },
  },
  extension: {
    instructions: {
      learnMoreUrl: 'https://my-wallet/learn-more',
      steps: [
        {
          description:
            'We recommend pinning My Wallet to your taskbar for quicker access to your wallet.',
          step: 'install',
          title: 'Install the My Wallet extension',
        },
        {
          description:
            'Be sure to back up your wallet using a secure method. Never share your secret phrase with anyone.',
          step: 'create',
          title: 'Create or Import a Wallet',
        },
        {
          description:
            'Once you set up your wallet, click below to refresh the browser and load up the extension.',
          step: 'refresh',
          title: 'Refresh your browser',
        },
      ],
    },
  },
  createConnector: getWalletConnectConnector({ projectId }),
});
```

จากนั้นคุณสามารถเพิ่มกระเป๋าสตางค์ที่กำหนดเองลงใน [รายการกระเป๋าสตางค์](/docs/custom-wallet-list) ของคุณได้
