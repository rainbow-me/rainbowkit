# BoutonPortefeuille

> Remarque : Le `BoutonPortefeuille` repose actuellement sur la norme de portefeuille EIP-1193, mais prendra en charge EIP-6963 dans un avenir proche.

Le nouveau composant `BoutonPortefeuille` aide les dApps avec des implémentations de liste de portefeuilles personnalisées à adopter RainbowKit et tous ses avantages sans maintenance.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

Comme le `BoutonConnexion`, le composant `BoutonPortefeuille.Custom` est disponible pour des implémentations et un style personnalisés.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

La plupart des dApps sont mieux servies par le [BoutonConnexion](https://www.rainbowkit.com/docs/connect-button). Consultez la documentation [ici](https://www.rainbowkit.com/docs/wallet-button) pour plus d'informations sur l'adoption et les cas d'utilisation du `BoutonPortefeuille`.
