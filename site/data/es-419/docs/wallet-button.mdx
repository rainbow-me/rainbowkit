# BotónBilletera

> Nota: Actualmente, el `BotónBilletera` se basa en el estándar de billetera EIP-1193, pero pronto admitirá el EIP-6963.

El nuevo componente `BotónBilletera` ayuda a las dApps con implementaciones personalizadas de listas de billeteras a adoptar RainbowKit y todos sus beneficios sin mantenimiento.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Al igual que el `ConnectButton`, el componente `BotónBilletera.Personalizado` está disponible para implementaciones y estilos personalizados.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

La mayoría de las dApps se benefician más al usar el [BotónConectar](https://www.rainbowkit.com/docs/connect-button). Consulta los [documentos aquí](https://www.rainbowkit.com/docs/wallet-button) para obtener más información sobre la adopción y casos de uso de `BotónBilletera`.
