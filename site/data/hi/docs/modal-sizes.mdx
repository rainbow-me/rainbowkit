# मोडल के आकार

## चौड़ा बनाम कॉम्पैक्ट आकार

डिफ़ॉल्ट रूप से, RainbowKit का `modalSize` `wide` पर सेट है (और ऐसा दिखता है):

<Img src="/modal-default.png" height="569" width="1029" />

हम `कॉम्पैक्ट` मोडल आकार भी प्रदान करते हैं, जो ऐसा दिखता है:

<Img src="/modal-compact.png" height="569" width="1029" />

अपने मोडल आकार को `compact` पर सेट करने के लिए, बस `modalSize="compact"` को अपने `RainbowKitProvider` में एक prop के रूप में जोड़ें:

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
