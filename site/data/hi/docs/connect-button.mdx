# ConnectButton

## ConnectButton का उपयोग करना और अनुकूलित करना

यह मुख्य घटक है। यह connect/disconnect बटन को रेंडर करने के लिए ज़िम्मेदार है, साथ ही साथ chain-swapping UI का भी।

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> नोट: सुनिश्चित करें कि आपका ऐप आवश्यक प्रदाताओं में लिपटा हुआ है। [अधिक पढ़ें](/docs/installation#wrap-providers).

### प्रॉप्स

`ConnectButton` घटक कई प्रॉप्स को उजागर करता है जिससे उसकी उपस्थिति को अनुकूलित किया जा सकता है, जिसमें विभिन्न तत्वों की दृश्यता को टॉगल करना शामिल है।

<PropsTable
  data={[
    {
      name: 'label',
      required: false,
      default: 'Connect Wallet',
      type: 'string',
      description: 'The text label displayed on the Connect button',
    },
    {
      name: 'accountStatus',
      required: false,
      default: 'full',
      type: '"avatar" | "address" | "full" | { smallScreen: AccountStatus, largeScreen?: AccountStatus }',

      description:
        "Whether the active account's avatar and/or address is displayed",
    },
    {

      typeSimple: 'enum',
      description:
        "Whether the current chain's icon and/or name is displayed, or hidden entirely",
    },
    {
      name: 'showBalance',
      required: false,
      default: '{ smallScreen: false, largeScreen: true }',

    },
  ]}
/>

> नोट: अंतर्निहित `largeScreen` ब्रेकपॉइंट `768px` है।

### उदाहरण

#### लेबल

कस्टम `ConnectButton` टेक्स्ट सेट करने के लिए `label` प्रॉप का उपयोग करें।

```tsx
<ConnectButton label="__Sign in__" />
```

#### खाता स्थिति

यहाँ `accountStatus` प्रॉप का उपयोग करने के कुछ अलग-अलग तरीके हैं।

केवल खाते का अवतार दिखाएँ।

```tsx
<ConnectButton accountStatus="__avatar__" />
```

केवल खाते का पता दिखाएँ।

```tsx
<ConnectButton accountStatus="__address__" />
```

#### चैन स्थिति

यहाँ `chainStatus` प्रॉप का उपयोग करने के कुछ अलग-अलग तरीके हैं।

केवल वर्तमान चैन का आइकॉन दिखाएँ।

```tsx
<ConnectButton chainStatus="__icon__" />
```

केवल वर्तमान चैन का नाम दिखाएँ।

```tsx
<ConnectButton chainStatus="__name__" />
```

चैन UI को पूरी तरह से छिपाएँ।

```tsx
<ConnectButton chainStatus="__none__" />
```

#### शेष राशि दिखाएँ

शेष राशि को छिपाने/दिखाने के लिए `showBalance` प्रॉप का उपयोग करें।

शेष राशि को छिपाएँ।

```tsx
<ConnectButton showBalance={__false__} />
```

#### रेस्पोंसिव

इन प्रॉप्स को रेस्पोंसिव प्रारूप में भी परिभाषित किया जा सकता है।

छोटी स्क्रीन पर, केवल खाता आइकन दिखाएँ। लेकिन बड़ी स्क्रीन पर, आइकन और पता दोनों दिखाएँ।

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

छोटी स्क्रीन पर, शेष राशि छिपाएँ। लेकिन बड़ी स्क्रीन पर, इसे दिखाएँ।

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
