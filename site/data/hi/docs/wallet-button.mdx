# वॉलेट बटन

> नोट: `वॉलेट बटन` वर्तमान में EIP-1193 वॉलेट मानक पर निर्भर करता है, लेकिन निकट भविष्य में EIP-6963 का समर्थन करेगा।

नया `वॉलेट बटन` कॉंपोनेंट dApps को कस्टम वॉलेट सूची के कार्यान्वयन के साथ RainbowKit को अपनाने और इसके सभी रखरखाव-मुक्त लाभों के साथ मदद करता है।

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

`कनेक्ट बटन` की तरह, `वॉलेट बटन.Custom` कॉंपोनेंट अनुकूलन कार्यान्वयन और स्टाइलिंग के लिए उपलब्ध है।

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

अधिकांश dApps के लिए [ConnectButton](https://www.rainbowkit.com/docs/connect-button) सबसे अच्छा विकल्प है। अधिक जानकारी के लिए `वॉलेट बटन` को अपनाने और उपयोग के मामलों के बारे में [यहां](https://www.rainbowkit.com/docs/wallet-button) देखिए।
