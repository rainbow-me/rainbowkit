# वॉलेटबटन

> नोट: `वॉलेटबटन` फिलहाल EIP-1193 वॉलेट मानक पर निर्भर है, लेकिन निकट भविष्य में EIP-6963 का समर्थन करेगा।

नया `वॉलेटबटन` कॉम्पोनेंट कस्टम वॉलेट सूची निष्पादन के साथ dApps की मदद करता है और RainbowKit के सभी रख-रखाव-मुक्त लाभों को अपनाता है।

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

`ConnectButton` की तरह, `वॉलेटबटन.Custom` कंपोनेंट कस्टम निष्पादन और स्टाइलिंग के लिए उपलब्ध है।

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

अधिकांश dApps के लिए [ConnectButton](https://www.rainbowkit.com/docs/connect-button) सर्वश्रेष्ठ कार्य करता है। `वॉलेटबटन` की स्वीकृति और उपयोग मामलों के बारे में अधिक जानकारी के लिए docs [यहाँ](https://www.rainbowkit.com/docs/wallet-button) देखें।
