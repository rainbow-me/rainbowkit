# BotãoCarteira

> Nota: O `BotãoCarteira` atualmente depende do padrão de carteiras EIP-1193, mas em breve suportará o EIP-6963.

O novo componente `BotãoCarteira` ajuda dApps com implementações de lista de carteiras personalizadas a adotarem o RainbowKit e todos os seus benefícios sem manutenção.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

Assim como o `BotãoConectar`, o componente `BotãoCarteira.Custom` está disponível para implementações e estilos personalizados.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

A maioria dos dApps são melhor atendidos pelo [BotãoConectar](https://www.rainbowkit.com/docs/connect-button). Consulte a documentação [aqui](https://www.rainbowkit.com/docs/wallet-button) para mais informações sobre a adoção e casos de uso do `BotãoCarteira`.
