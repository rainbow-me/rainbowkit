# 彈出窗口尺寸

## 寬闊與緊湊尺寸

預設情況下，RainbowKit 的 `modalSize` 設定為 `wide`（看起來像這樣）：

<Img src=\"/modal-default.png\" height=\"569\" width=\"1029\" />

我們還提供一個 `compact` 模式尺寸，看起來像這樣：

<Img src=\"/modal-compact.png\" height=\"569\" width=\"1029\" />

若要將您的模組尺寸設為 `compact`，只需將 `modalSize=\"compact\"` 作為屬性添加到您的 `RainbowKitProvider` 中：

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
