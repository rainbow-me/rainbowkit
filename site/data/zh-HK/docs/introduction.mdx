# RainbowKit

## 最佳嘅方式去連接錢包 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一個 [React](https://reactjs.org/) 庫，使您輕鬆在您的 dApp 上添加錢包連接。它直觀、響應迅速且可定制。

### 功能

#### 錢包管理

為您的 dApp 提供現成的錢包管理。除了處理錢包的連接和斷開外，RainbowKit 還支持多種錢包、交換鏈、解析地址到 ENS、顯示餘額等！

#### 可定制

您可以調整 RainbowKit 的 UI 以匹配您的品牌。您可以從一些預定的主色調和邊框半徑配置中進行選擇。對於更高級的用例，您可以提供完全自訂的主題，渲染您自己的按鈕並省略某些功能。也支持深色模式。

#### 行業標準

為了更好的互操作性，我們依靠 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/)——這是空間中最常用的庫。

像 EIP-1193 和 EIP-6963 的錢包標準支持開箱即用，使瀏覽器錢包用戶更輕鬆地將錢包無縫連接到您的 dApp。
