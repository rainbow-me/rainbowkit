# RainbowKit

## 最佳嘅方式去連接錢包 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一個 [React](https://reactjs.org/) 函式庫，使將錢包連接加入到您的 dapp 中變得輕而易舉。 它直觀、響應快速且可自訂。

### 功能

#### 錢包管理

現成的錢包管理功能，可用於您的 dapp。 除了處理錢包的連接和斷開，RainbowKit 還支持多個錢包、互換連接鏈、將地址解析為 ENS、顯示餘額等等功能！

#### 可自訂

您可以調整 RainbowKit 的 UI 以匹配您的品牌形象。 您可以從幾個預設的顏色和邊框半徑配置中進行選擇。 對於更高級的使用情境，您可以提供完全自訂的主題，渲染您自己的按鈕並省略某些功能。 包括深色模式。

#### 行業標準

為了更好地與大多數產品互通，我們依賴於 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/) —— 這個領域中最常用的函式庫。

像 EIP-1193 和 EIP-6963 這樣的錢包標準在現成的支持，使瀏覽器錢包用戶更輕松地將其錢包無縫連接到您的 dApp。
