# 錢包按鈕

> 注意：`WalletButton` 目前依賴於 EIP-1193 錢包標準，但將在不久的將來支持 EIP-6963。

新的 `WalletButton` 組件幫助 dApps 採用自訂錢包列表實現及 RainbowKit 所有免維護的優勢。

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

就像 `ConnectButton`，`WalletButton.Custom` 組件適用於自訂實現和樣式設置。

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

大多數 dApps 最佳選擇是 [ConnectButton](https://www.rainbowkit.com/docs/connect-button)。參考 [這裡](https://www.rainbowkit.com/docs/wallet-button) 的文檔以獲得更多關於 `WalletButton` 採用和使用案例的信息。
