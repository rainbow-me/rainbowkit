# 炫酷模式

## 啟用炫酷模式 🌈🌈🌈

你可以通過 `RainbowKitProvider` 的 `coolMode` 屬性來啟用炫酷模式。

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 但這是什麼？

當從錢包列表中選擇一個錢包時，會出現表情符號爆炸 🌈🌈🌈。 它會使用你選擇的錢包圖標。
