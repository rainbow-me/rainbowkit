# 連接按鈕

## 使用和自訂連接按鈕

這是主要組件。 它負責渲染連接/斷開按鈕，以及鏈切換用戶端。

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> 注意：請確保您的應用程序被包裝在必要的提供者中。[閱讀更多](/docs/installation#wrap-providers)。

### 屬性

`ConnectButton` 組件提供多個屬性來自訂其外觀，包括切換不同元素的可見性。

<PropsTable

    },
    {
      name: 'accountStatus',
      required: false,
      default: 'full',
      type: '"avatar" | "address" | "full" | { smallScreen: AccountStatus, largeScreen?: AccountStatus }',
      typeSimple: 'enum',
      description:
        "是否顯示活動帳戶的頭像和/或地址",
    },
    {
      name: 'chainStatus',
      required: false,
      default: '{ smallScreen: "icon", largeScreen: "full" }',
      type: '"icon" | "name" | "full" | "none" | { smallScreen: ChainStatus, largeScreen?: ChainStatus }',
      typeSimple: 'enum',
      description:
        "是否顯示或完全隱藏當前鏈的圖示和/或名稱",
    },
    {
      name: 'showBalance',
      required: false,
      default: '{ smallScreen: false, largeScreen: true }',
      type: 'boolean | { smallScreen: boolean, largeScreen?: boolean }',
      typeSimple: 'enum',
      description:
        '餘額是否顯示在帳戶名稱旁邊',
    },
  ]}
/>

> 注意：內建的 `largeScreen` 斷點為 `768px`。

### 示例

#### 標籤

使用 `label` 屬性來設置自訂 `ConnectButton` 文本。

```tsx
<ConnectButton label="__Sign in__" />
```

#### 帳戶狀態

這裡有一些不同方法來使用 `accountStatus` 屬性。

僅顯示帳戶的頭像。

```tsx
<ConnectButton accountStatus="__avatar__" />
```

僅顯示帳戶的地址。

```tsx
<ConnectButton accountStatus="__address__" />
```

#### 鏈狀態

這裡有一些不同的方法來使用 `chainStatus` 屬性。

僅顯示當前鏈的圖示。

```tsx
<ConnectButton chainStatus="__icon__" />
```

僅顯示當前鏈的名稱。

```tsx
<ConnectButton chainStatus="__name__" />
```

完全隱藏鏈用戶介面。

```tsx
<ConnectButton chainStatus="__none__" />
```

#### 顯示餘額

使用 `showBalance` 屬性來隱藏/顯示餘額。

隱藏餘額。

```tsx
<ConnectButton showBalance={__false__} />
```

#### 響應式設計

這些屬性也可以定義為響應格式。

在小螢幕上，僅顯示帳戶圖示。但在大螢幕上，顯示圖示和地址。

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

在小螢幕上，隱藏餘額。但在大螢幕上，顯示它。

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
