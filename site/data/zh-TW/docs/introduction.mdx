# RainbowKit

## 連接錢包的最佳方式 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一個 [React](https://reactjs.org/) 庫，使您輕鬆地為 dapp 添加錢包連接功能。 它直觀、響應迅速且可自定義。

### 功能

#### 錢包管理

為您的 dapp 提供現成的錢包管理。 除了處理錢包的連接和斷開連接外，RainbowKit 還支持眾多錢包，交換連接鏈，解析地址到 ENS，顯示餘額等更多功能！

#### 可自定義

您可以調整 RainbowKit 的 UI 以符合您的品牌。 您可以從一些預定義的重點顏色和邊框半徑配置中選擇。 對於更高級的使用情況，您可以提供完全自定義的主題，渲染您自己的按鈕並省略某些功能。 包含深色模式。

#### 行業標準

為了更好地與大多數產品互操作，我們依賴於 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/)——該領域中最常用的庫。

像 EIP-1193 和 EIP-6963 這樣的錢包標準支持現成開箱即用，使得瀏覽器錢包用戶比以往更容易無縫連接其錢包到您的 dApp。
