# RainbowKit

## 連接錢包的最佳方式 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit 是一個 [React](https://reactjs.org/) 庫，它讓您可以輕鬆地將錢包連接添加到您的 dApp。它直觀、響應迅速且可自定義。

### 特性

#### 錢包管理

為您的 dApp 提供開箱即用的錢包管理。除了處理錢包的連接和斷開之外，RainbowKit 支援多種錢包，交換連接鏈，將地址解析為 ENS，顯示餘額等更多功能！

#### 可自定義

您可以調整 RainbowKit 的 UI 以匹配您的品牌。您可以從幾個預定義的強調顏色和邊框半徑配置中進行選擇。對於更高級的用法，您可以提供完整的自定義主題，呈現您自己的按鈕並省略某些功能。包括暗模式。

#### 行業標準

為了更好地與大多數產品互操作，我們依賴於 [viem](https://viem.sh) 和 [wagmi](https://wagmi.sh/) — 在該領域中最常用的庫。

像 EIP-1193 和 EIP-6963 這樣的錢包標準開箱即用，讓瀏覽器錢包用戶比以往更加輕鬆地無縫連接他們的錢包到您的 dApp。
