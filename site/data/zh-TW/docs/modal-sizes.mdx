# 模態框大小

## 寬與窄的大小

默認情況下，RainbowKit 的 `modalSize` 設置為 `wide`（顯示如下）:

<Img src="/modal-default.png" height="569" width="1029" />

我們還提供一個 `compact` 形式的模態框大小，顯示如下：

<Img src="/modal-compact.png" height="569" width="1029" />

要將您的模態框大小設置為 `compact`，只需在 `RainbowKitProvider` 中添加 `modalSize="compact"` 作為屬性：

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
