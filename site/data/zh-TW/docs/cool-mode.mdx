# 酷模式

## 啟用酷模式 🌈🌈🌈

您可以在 `RainbowKitProvider` 上通過 `coolMode` 屬性啟用酷模式。

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 但這是什麼？

選擇錢包列表中的錢包時，表情符號爆炸 🌈🌈🌈。 它會使用您選擇的錢包圖標。
