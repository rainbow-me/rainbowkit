# 酷炫模式

## 啟用酷炫模式 🌈🌈🌈

您可以通過 `RainbowKitProvider` 的 `coolMode` prop 來啟用酷炫模式。

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 但那是什麼呢？

當從錢包列表中選擇一個錢包時，表情符號爆炸 🌈🌈🌈。它用您選擇的錢包圖示。
