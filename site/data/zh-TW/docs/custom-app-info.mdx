# 自定義應用程式資訊

## 自定義應用程式的資訊

您可以在 `RainbowKitProvider` 中的 `appInfo` 屬性傳遞應用程式的資訊。您可以修改的屬性包括應用程式名稱（`appName`）和「了解更多」按鈕在連接模態中的重定向連結（`learnMoreUrl`）：

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'Rainbowkit Demo',
        learnMoreUrl: 'https://learnaboutcryptowallets.example',
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```

### 新增聲明

您可以提供自己的聲明以顯示在連接模態的底部。在 `appInfo` 中的 `disclaimer` 屬性接受具備 `Text` 和 `Link` 參數的 `DisclaimerComponent`。您可以使用它來構建繼承自定義主題系統的聲明。我們為 TypeScript 使用者匯出 `DisclaimerComponent` 類型：

```tsx
import {
  RainbowKitProvider,
  DisclaimerComponent,
} from '@rainbow-me/rainbowkit';

const Disclaimer: DisclaimerComponent = ({ Text, Link }) => (
  <Text>
    By connecting your wallet, you agree to the{' '}
    <Link href="https://termsofservice.xyz">Terms of Service</Link> and
    acknowledge you have read and understand the protocol{' '}
    <Link href="https://disclaimer.xyz">Disclaimer</Link>
  </Text>
);

const App = () => {
  return (
    <RainbowKitProvider
      appInfo={{
        appName: 'RainbowKit Demo',
        disclaimer: Disclaimer,
      }}
      {...etc}
    >
      {/* ... */}
    </RainbowKitProvider>
  );
};
```
