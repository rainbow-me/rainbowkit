# 連接按鈕

## 使用及自定義連接按鈕

這是主要的組件。 它負責渲染連接/斷開按鈕及切換鏈的介面。

```tsx
import { ConnectButton } from '@rainbow-me/rainbowkit';

export const YourApp = () => {
  return <ConnectButton />;
};
```

> 注意：確保您的應用程式包裝在必要的提供者中。 [閱讀更多](/docs/installation#wrap-providers).

### 屬性

`ConnectButton` 組件暴露了幾個屬性用於自定義其外觀，包括切換不同元素的可見性。

<PropsTable
  data={[
    {
      name: 'label',
      required: false,
      default: '連接錢包',
      type: 'string',
      description: '顯示在連接按鈕上的文字標籤',
    },
    {
      name: 'accountStatus',
      required: false,
      default: 'full',
      type: '"avatar" | "address" | "full" | { smallScreen: AccountStatus, largeScreen?: AccountStatus }',
      typeSimple: 'enum',
      description:
        "是否顯示活躍帳戶的頭像和/或地址",
    },
    {
      name: 'chainStatus',
      required: false,
      default: '{ smallScreen: "icon", largeScreen: "full" }',
      type: '"icon" | "name" | "full" | "none" | { smallScreen: ChainStatus, largeScreen?: ChainStatus }',
      typeSimple: 'enum',
      description:
        "是否顯示或完全隱藏當前鏈的圖標和/或名稱",
    },
    {
      name: 'showBalance',
      required: false,
      default: '{ smallScreen: false, largeScreen: true }',
      type: 'boolean | { smallScreen: boolean, largeScreen?: boolean }',
      typeSimple: 'enum',
      description:
        '是否在帳戶名稱旁顯示餘額',
    },
  ]}
/>

> 注意：內置的`largeScreen`斷點是`768px`。

### 範例

#### 標籤

使用`label`屬性設置自定義`ConnectButton`文本。

```tsx
<ConnectButton label="__Sign in__" />
```

#### 帳戶狀態

以下是幾種不同的方式來使用`accountStatus`屬性。

只顯示帳戶的頭像。

```tsx
<ConnectButton accountStatus="__avatar__" />
```

只顯示帳戶的地址。

```tsx
<ConnectButton accountStatus="__address__" />
```

#### 鏈狀態

以下是幾種不同的方式來使用`chainStatus`屬性。

只顯示當前鏈的圖標。

```tsx
<ConnectButton chainStatus="__icon__" />
```

只顯示當前鏈的名稱。

```tsx
<ConnectButton chainStatus="__name__" />
```

完全隱藏鏈的介面。

```tsx
<ConnectButton chainStatus="__none__" />
```

#### 顯示餘額

使用`showBalance`屬性來隱藏/顯示餘額。

隱藏餘額。

```tsx
<ConnectButton showBalance={__false__} />
```

#### 響應式

這些屬性也可以定義為響應式格式。

在小屏幕上，只顯示帳戶圖標。 但在大屏幕上，顯示圖標和地址。

```tsx
<ConnectButton
  accountStatus={{
    smallScreen: 'avatar',
    largeScreen: 'full',
  }}
/>
```

在小屏幕上，隱藏餘額。 但在大屏幕上，顯示餘額。

```tsx
<ConnectButton
  showBalance={{
    smallScreen: false,
    largeScreen: true,
  }}
/>
```
