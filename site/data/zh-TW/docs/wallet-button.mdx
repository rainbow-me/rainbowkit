# WalletButton

> 注意：目前 `WalletButton` 依賴於 EIP-1193 錢包標準，但很快將支持 EIP-6963。

新的 `WalletButton` 元件協助 dApps 實施自定義的錢包列表，以採用 RainbowKit 及其無需維護的所有優勢。

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

如同 `ConnectButton`，`WalletButton.Custom` 元件可用於自定義實現和樣式。

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

大多數 dApps 最適合使用 [ConnectButton](https://www.rainbowkit.com/docs/connect-button)。 在[這裡](https://www.rainbowkit.com/docs/wallet-button)參考檔，以瞭解更多關於 `WalletButton` 採用和用例的資訊。
