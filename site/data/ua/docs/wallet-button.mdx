# WalletButton

> Примітка: `WalletButton` наразі спирається на стандарт гаманця EIP-1193, але в найближчому майбутньому буде підтримувати EIP-6963.

Новий компонент `WalletButton` допомагає dApps з кастомними реалізаціями списку гаманців прийняти RainbowKit і всі його переваги без необхідності обслуговування.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

Як і `ConnectButton`, компонент `WalletButton.Custom` доступний для користувацьких реалізацій та стилізації.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Більшість dApps найкраще працюють за допомогою [ConnectButton](https://www.rainbowkit.com/docs/connect-button). Дивіться документацію [тут](https://www.rainbowkit.com/docs/wallet-button) для отримання додаткової інформації про впровадження та випадки використання `WalletButton`.
