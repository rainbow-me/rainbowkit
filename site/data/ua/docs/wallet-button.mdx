# Кнопка Гаманця

> Примітка: `Кнопка Гаманця` наразі покладається на стандарт гаманця EIP-1193, але в найближчому майбутньому підтримає EIP-6963.

Новий компонент `Кнопка Гаманця` допомагає dApps з реалізацією нестандартного списку гаманців прийняти RainbowKit і всі його переваги без обслуговування.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="baseAccount" />
```

Як і `ConnectButton`, компонент `WalletButton.Custom` доступний для кастомних реалізацій та стилізації.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

Більшість dApps найкраще обслуговуються [ConnectButton](https://www.rainbowkit.com/docs/connect-button). Звертайтесь до документації [тут](https://www.rainbowkit.com/docs/wallet-button) для отримання додаткової інформації про впровадження та випадки використання `Кнопки Гаманця`.
