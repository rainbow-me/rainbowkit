# 모달 크기

## 넓은 vs. 컴팩트 크기

기본적으로, RainbowKit의 `modalSize`는 `wide`로 설정되며, 다음과 같은 모양입니다:

<Img src="/modal-default.png" height="569" width="1029" />

우리는 또한 다음과 같은 `compact` 모달 크기를 제공합니다:

<Img src="/modal-compact.png" height="569" width="1029" />

모달 크기를 `compact`로 설정하려면 `RainbowKitProvider`에 `modalSize="compact"` 속성을 추가하세요:

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

export const App = () => (
  <RainbowKitProvider modalSize="compact" {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```
