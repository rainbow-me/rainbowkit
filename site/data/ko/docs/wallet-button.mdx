# 지갑 버튼

> 참고: `WalletButton`은 현재 EIP-1193 지갑 표준에 의존하고 있지만, 가까운 미래에 EIP-6963을 지원할 것입니다.

새로운 `WalletButton` 구성 요소는 맞춤형 지갑 목록 구현을 통해 dApp이 RainbowKit 및 그 유지 관리가 필요 없는 모든 이점을 채택할 수 있도록 도와줍니다.

```tsx
import { WalletButton } from '@rainbow-me/rainbowkit';

<WalletButton wallet="rainbow" />
<WalletButton wallet="metamask" />
<WalletButton wallet="coinbase" />
```

`ConnectButton`처럼, `WalletButton.Custom` 구성 요소는 맞춤 구현 및 스타일링을 위해 제공됩니다.

```tsx
<WalletButton.Custom wallet="rainbow">
  {({ ready, connect }) => {
    return (
      <button
        type="button"
        disabled={!ready}
        onClick={connect}
      >
        Connect Rainbow
      </button>
    );
  }}
</WalletButton.Custom>
```

대부분의 dApp은 [ConnectButton](https://www.rainbowkit.com/docs/connect-button)에 의해 가장 잘 제공됩니다. `WalletButton` 채택 및 사용 사례에 대한 추가 정보는 [여기](https://www.rainbowkit.com/docs/wallet-button)를 참조하세요.
