# 멋진 모드

## 멋진 모드 활성화 🌈🌈🌈

`RainbowKitProvider`의 `coolMode` 속성을 통해 멋진 모드를 활성화할 수 있습니다.

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 그런데 이게 뭐죠?

지갑 목록에서 지갑을 선택하면 이모티콘 폭발이 일어납니다 🌈🌈🌈. 선택한 지갑의 아이콘을 사용합니다.
