# 쿨 모드

## 쿨 모드 활성화 🌈🌈🌈

`RainbowKitProvider`의 `coolMode` 속성을 통해 쿨 모드를 활성화할 수 있습니다.

```tsx
import { RainbowKitProvider } from '@rainbow-me/rainbowkit';

const App = () => (
  <RainbowKitProvider coolMode {...etc}>
    {/* Your App */}
  </RainbowKitProvider>
);
```

### 하지만 그것이 무엇인가요?

지갑 목록에서 지갑을 선택할 때 이모지 폭발 🌈🌈🌈. 선택한 지갑의 아이콘을 사용합니다.
