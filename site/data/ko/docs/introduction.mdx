# RainbowKit

## 지갑을 연결하는 최고의 방법 🌈

<Video src="https://res.cloudinary.com/rainbowme/video/upload/s--aTmEjSrr--/f_auto:video,q_auto/v1/rainbowkit/site-assets/rainbowkit-video-cutdown-edit_gy0q16" />

RainbowKit은 지갑 연결을 당신의 dapp에 쉽게 추가할 수 있게 하는 [React](https://reactjs.org/) 라이브러리입니다. 직관적이고 반응성이 좋고 맞춤 설정이 가능합니다.

### 특징

#### 지갑 관리

당신의 dapp를 위한 풀 패키지 지갑 관리. 지갑의 연결과 연결 해제를 처리하는 것 외에도, RainbowKit은 다양한 지갑을 지원하고, 연결 체인을 교환하고, 주소를 ENS로 해석하고, 잔액을 표시하고 더 많은 작업을 수행합니다!

#### 맞춤 설정 가능

RainbowKit UI를 귀하의 브랜드에 맞게 조정할 수 있습니다. 미리 정의된 몇 가지 강조 색상과 테두리 반경 구성에서 선택할 수 있습니다. 더 복잡한 사용 사례의 경우, 완전히 사용자 정의된 테마를 제공하거나, 자신만의 버튼을 렌더링하거나, 특정 기능을 생략할 수 있습니다. 다크 모드가 포함되어 있습니다.

#### 산업 표준

대부분의 제품과의 더 나은 상호 운용성을 위해, [viem](https://viem.sh)과 [wagmi](https://wagmi.sh/)인 가장 널리 사용되는 라이브러리를 사용하고 있습니다.

지갑 표준인 EIP-1193 및 EIP-6963은 기본적으로 지원되므로, 브라우저 지갑 사용자가 자신의 지갑을 dApp에 원활하게 연결할 수 있게 되었습니다.
